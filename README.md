# My Journal - Website

This is the official website for the My Journal iOS app - a personal journaling and reflection application designed to help users track their personal growth through daily reflection questions.

## Features

- **Daily Reflection Questions**: Thought-provoking questions for personal growth
- **30-Day Growth Cycles**: Track development over monthly cycles
- **Previous Answer Review**: Compare current thoughts with past entries
- **Passcode Protection**: Secure your personal thoughts
- **Auto-Save & Sync**: Never lose your reflections
- **Beautiful Interface**: Clean, paper-like design

## Website Structure

- `index.html` - Main landing page with features and download information
- `styles.css` - Styling and responsive design
- `script.js` - Interactive functionality and mobile navigation
- `privacy.html` - Privacy policy
- `terms.html` - Terms of service
- `assets/` - App icons and images
